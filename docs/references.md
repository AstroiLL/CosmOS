# References

## NotebookLM

### Agent OS

В NotebookLM есть пример готовой агентной ОС: **Agent OS**.

Использовать как референс при проектировании CosmOS:

- архитектура агентной ОС;
- организация интерфейсов;
- подход к памяти;
- UX для управления агентами и задачами;
- идеи для MVP и roadmap.

Локальная заметка: перед проектированием крупных частей CosmOS стоит сверяться с этим NotebookLM-ноутбуком.
