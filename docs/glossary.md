# Glossary

- **CosmOS** — агентная ОС/оркестратор.
- **Kernel** — ядро маршрутизации задач и политик.
- **Adapter** — слой подключения конкретного CLI/агента.
- **Interface** — способ общения с CosmOS: CLI, Web, Telegram, API.
- **Memory backend** — источник/хранилище памяти: Obsidian, SQLite, Mem0, vector DB.
- **Skill** — процедурная память: повторяемый способ выполнения задач.
